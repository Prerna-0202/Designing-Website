Downloads

Start Photoshop Plugin

To import designs from Photoshop, you'll need to first create a project inside the plugin or Mockplus. Inside the plugin, click the triangle icon next to the name of the project and click "+" in the dropdown menu to create a project.When you have a project created, you can then specify a target page group to import designs you need

Import Designs from Figma

With the Figma plugin, apart from exporting designs to Mockplus Cloud, you can also mark layers or groups as assets for export. Select the layers or groups you'd like to export as assets and click "Add slide layer". Those layers and groups will be uploaded to Mockplus Cloud along with your designs. And then, these asset details are available for your developers and other team members to view online.

Design Tool Integration

In Sketch, you can easily add any assets to your design system libraries on the web using our Sketch plugin. Download and install the Sketch plugin first. And then, open your Sketch app and start the plugin in this path: Plugins> Mockplus DS.Create a new group in the "Colors" category; Select a color layer in Sketch and click the "+" icon at the top right corner of the plugin.

Import Designs from Adobe XD

You can import assets and designs directly from Adobe XD into Mockplus using our plugin. Inside the plugin, you can easily mark which layers you want as assets to be imported into Mockplus with your designs.Mockplus Cloud offers users a plugin to export artboards with slices marked from Adobe XD quickly and directly.
